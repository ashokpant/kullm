# KULLM

It contains LLM use cases and api to access them. It also has demo UI.

## Features

1. Surname classification

## Dependencies

Use poetry

```bash 
poetry install
```

Add new dependency

```bash
poetry add torch
```

## Build

```bash
pip install -r requirements.txt
```

## Run API

```bash
export PYTHONPATH=`pwd`
python kullm/main.py
```


## Run UI

```bash
streamlit run ui/app.py
```
